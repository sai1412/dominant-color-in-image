Algorithm used

K-Means Clustering Algorithm

Defination

It is an iterative algorithm that divides the unlabeled dataset into k different clusters in such a way that each dataset belongs only one group that has similar properties.

STEP-1:

Select the number K to decide the number of clusters and Select random K points or centroids. (It can be other from the input dataset).

STEP-2:

Assign each data point to their closest centroid, which will form the predefined K clusters.

STEP-3:

Calculate the variance and place a new centroid of each cluster.

STEP-4:

Repeat the second steps, which means reassign each datapoint to the new closest centroid of each cluster.

STEP-5:

If any reassignment occurs, then go to step-4 else go to FINISH.
The model is ready.
